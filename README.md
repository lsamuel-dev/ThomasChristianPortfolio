# User Stories

## As a Visitor 50+

- I want to immediately understand what Thomas Christian offers
- So that I can decide whether to explore or subscribe

**Acceptance Criteria:**

- Given I land on Home
- When the page loads
- Then I see a hero intro, a carousel of ebooks Thomas has written with brief captions and “Get/View” buttons

## As a Prospect evaluating a product

- I want a simple product card with benefits and a sample
- So that I can feel safe to buy

**Acceptance Criteria:**

- Given I open the ebook section
- When the page loads
- Then I see the cover image, 3 bullet benefits, price, a “Preview PDF,” and a “Buy on Gumroad/Amazon” button

## As a Hesitant Visitor

- I want a newsletter/freebie opt-in
- So that I can try before I buy

**Acceptance Criteria:**

- Given I scroll to mid-page
- When I see the signup form
- Then I can enter my email and get a “Free Chapter / Cheat Sheet” confirmation

## As a Visitor wanting proof

- I want credibility cues or proof
- So that I can trust the brand

**Acceptance Criteria:**

- Given I view the "About/Trust" section
- When there are no sales yet
- Then I see the author bio, mission, “Featured on” logos (YouTube, Medium, Gumroad, Amazon), and space for future testimonials

## As a Mobile User

- I want a clear navbar with a hamburger menu
- So that I can reach sections quickly

**Acceptance Criteria:**

- Given I'm on a small screen
- When I tap the hamburger
- Then I see links: Home, Ebooks, Prompt Bundles, Articles, Contact
